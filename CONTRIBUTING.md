# Contributing to Awesome Agent Orchestration Platforms

Thank you for your interest in contributing! We aim to make this the most comprehensive and up-to-date index for AI agent orchestration.

## Guidelines

- **Active Projects only:** Only submit tools that are actively maintained and have a clear "last updated" date within the last 6 months.
- **Fact-Based Descriptions:** Keep descriptions concise (1-2 sentences) and focused on features/use cases. Avoid marketing fluff.
- **Categorization:** Ensure your tool fits into one of the existing categories. If it requires a new category, please explain why in your PR.
- **Keyword Alignment:** When adding descriptions, use relevant terms like "Multi-Agent," "Orchestration," "Agentic Workflows," or "MCP" to help with discovery.

## How to add a new tool

1. Fork the repository.
2. Edit `README.md` to add your entry in the correct section.
3. Update `llms.txt` if the tool is a major framework.
4. Submit a Pull Request.

## Star the Repo
If you find this list useful, please star the repository to help others find it!
